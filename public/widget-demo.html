sBTC Payment Widget Demo

Interactive examples of the sBTC payment drop-in widgets

Demo Note: This demo uses test API keys. In production, replace with your actual public API key from the dashboard.

B Button Widget

The most common widget type. Perfect for product pages, shopping carts, and call-to-action sections.

<script src="/widget.js" data-sbtc-key="pk_test_demo123"></script> <div data-sbtc-button data-amount="50000" data-description="Premium Coffee Subscription"> </div>

I Inline Widget

A card-style widget that shows payment details inline. Great for checkout pages and payment forms.

<script src="/widget.js" data-sbtc-key="pk_test_demo123"></script> <div data-sbtc-inline data-amount="125000" data-description="MacBook Pro 16-inch" data-button-text="Pay with sBTC"> </div>

L Link Widget

A simple text link that triggers the payment modal. Ideal for minimal designs and text-heavy content.

You can donate here to support our project.
<script src="/widget.js" data-sbtc-key="pk_test_demo123"></script> You can <span data-sbtc-link data-amount="25000" data-description="Donate to Open Source">donate here</span> to support our project.

C Configuration Options

All available data attributes for customizing your widgets.

<!-- All configuration options --> <div data-sbtc-button data-amount="100000" <!-- Amount in microsBTC --> data-description="Product Name" <!-- Payment description --> data-button-text="Buy Now" <!-- Custom button text --> data-currency="sbtc" <!-- Currency (sbtc/btc) --> data-theme="blue" <!-- Theme variant --> data-success-url="/success" <!-- Redirect on success --> data-cancel-url="/cancel"> <!-- Redirect on cancel --> </div>
